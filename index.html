PRIMEIRO ACESSO

Entrando na plataforma web

Para entrar na plataforma, você vai precisar dos seguintes dados de acesso: login (usuário), senha e chave de sua plataforma.

Link para acesso a plataforma:

www.fp3d.com.br

MENU - CADASTRO

USUÁRIOS

O menu usuário, serve para cadastrar os usuários e gerenciar as permissões de otimizador e relatório na plataforma, lembrando que após efetuar o cadasto do usuário, ele deve ser inserido em um grupo de permissão, para ter seu devido acesso a plataforma.

MENU - PERMISSÕES

GRUPOS / USUÁRIOS X PERMISSÕES

O menu GRUPOS / USUÁRIOS X PERMISSÕES, Serve para você gerenciar as permissões de acesso a plataforma.

Aqui você cadastra o grupo, vincula as permissões de acesso desejada para esse grupo, em seguida, vincula o usuário ao grupo de permissões, assim todos os usuários contidos em um determinado grupo terão as mesmas permissões

MENU - CADASTRO

TEXTURA

É no menu TEXTURA, que cadastramos os materiais que iremos utilizar para aplicar nos modelos do projeto, como por exemplo: as chpas de mdf, vidros etc.

MENU - CADASTRO

PRODUTO

No menu produto, iremos cadastrar chapas e bordas e criar uma relação com a textura cadastrada, para assim poder gerar orçamento, plano de corte e relatórios.

Nós também vamos poder cadastrar, itens que não compõe o plugin, itens que ficarão disponíveis apenas para inserção no orçamento sem modelagem.

MENU - CADASTRO

MARGEM FABRICA/LOJA

No menu produto, iremos escolher a opção tipo de produto. para configurar as margens de fábrica e loja, devemos acessar o menu cadastro > produto > tipo de produto

Ao acessar, será exibido todos os tipos de produtos para inserção de margem de lucro de fábrica e loja.

A margem fabrica será calculado em cima do valor custo do produto de todos os produtos do tipo escolhido, a margem loja é calculado em cima do valor de fábrica, trazendo em seu orçamento o valor total, calculando custo + margem fábrica + margem loja.

MENU - COMERCIAL

AMBIENTE X MARGEM

A opção ambiente x margem, é usada para adicionar uma margem de lucro adicional no setor comercial, no preço final do orçamento, e assim poder diversificar seu orçamento, veja como funciona essa configuração:

Após ter cadastrado as margens de fábrica, loja será adicionado essa margem adicional em cima desse valor total, exemplo:

Valor dos produtos cadastrado no tipo mdf = R$ 1000,00 + 10 % margem fabrica = R$ 1.100,00 + 10 % margem loja = R$ 1.210,00 + ambiente x margem ( margem 1 = 10% ) = R$ 1.331,00 + ambiente x margem ( margem 2 = 10% ) = R$ 1.464,10, o valor final seria R$ 1.464,10.

Em uma formula matemática ficaria assim: Custo + % margem fabrica + % margem loja + % ambiente margem 1 + % ambiente margem 2

MENU - COMERCIAL

CLIENTE

Aqui você faz o cadastro de seus clientes, para poder gerar pedido e ou contrato, para orçamento rápido não será necessário, apenas para fechamento de pedido.

MENU - COMERCIAL

ORÇAMENTO

No menu orçamento, é possível realizar orçamentos pela web ou pelo Sketchup utilizando o plugin FP3D , veja também como emitir pedido, classificar parcelas e imprimir para seu cliente.

MENU - RELATÓRIO

RELATÓRIOS

Os relatórios de insumos ou materiais, são utilizados para listar tudo que você precisa comprar para produzir um determinado pedido, ele calcula todo material utilizado no pedido e exibe pra você, separado por categoria, como: MDF, VIDRO, FERRAGENS, ARAMADOS etc..

MENU - OTIMIZADORES

OTIMIZADORES

No menu otimizadores, você configura todos os parâmetros exportação do plano de corte para o seu otimizador padrão, nesse vídeo demonstramos com Corte-Certo e MaxCut, que são os otimizadores padrões que utilizamos na plataforma, más não são os únicos, nós integramos com qualquer otimizador do mercado, veja no vídeo algumas configurações dos otimizadores que já integramos.
